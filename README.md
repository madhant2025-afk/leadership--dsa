# Leaderboard System using AVL Tree in C

## 📌 Description

This project implements a dynamic leaderboard system using an AVL Tree.
It supports efficient score updates, ranking, and retrieval of top-K players.

## 🚀 Features

* Add Player
* Update Score
* Get Rank
* Top-K Players
* Display Leaderboard

## 🧠 Data Structure Used

* AVL Tree (Self-balancing Binary Search Tree)

## ⏱ Time Complexity

* Insert: O(log n)
* Update: O(log n)
* Rank: O(log n)
* Top-K: O(k + log n)

## ▶️ How to Run

1. Compile: `gcc leaderboard.c`
2. Run: `./a.out`
